# Construst

Plateforme statique interactive pour Construst.

## Lancer en local

```bash
cd /Users/mac/Documents/Construst
python3 -m http.server 4173
```

Puis ouvrir `http://127.0.0.1:4173`.

## Deployer sur GitHub Pages

1. Cree un repository GitHub.
2. Pousse ce dossier sur la branche `main`.
3. Le workflow GitHub Actions deploiera automatiquement le site sur GitHub Pages.
4. Dans GitHub:
   `Settings` > `Pages` > `Build and deployment` > `Source: GitHub Actions`

## Fichiers principaux

- `index.html`
- `styles.css`
- `app.js`
